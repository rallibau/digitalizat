Signatura d'un document PDF amb la configuració estandard (signatura a la primera pàgina i visible)
Certificats del magatzem de Windows.
El paràmetre d'entrada és la url del document que l'applet descarregarà per a signar.
Es guardaran els pdfs signats a una carpeta nova amb nom CATCert dins la carpeta personal de l'usuari.
No s'ha definit ni el color de fons ni el logo, i per tant es mostren els valors per defecte (fons blanc i logo CATCert).
L'applet es crida utilitzant un botó html que utilitzarà Javascript per invocar-lo.
