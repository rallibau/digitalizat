Signatura XAdES-BES detached.
Certificats del magatzem de Windows.
El paràmetre d'entrada són 3 resums (hash) pre-calculats dels documents a signar (veure codi font).
La sortida és via event Javascript. Es rep un event per a la signatura de cada document i un event final amb totes les signatures.
No s'ha definit ni el color de fons ni el logo, i per tant es mostren els valors per defecte (fons blanc i logo CATCert).
L'applet es crida utilitzant un botó html que utilitzarà Javascript per invocar-lo.
